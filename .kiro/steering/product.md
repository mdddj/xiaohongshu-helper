# Product Overview

**Xiaohongshu Helper** (启航发布) is a desktop application for content creators on Xiaohongshu (Little Red Book), China's popular lifestyle and social commerce platform.

## Core Purpose

Streamline content creation and publishing workflows for Xiaohongshu creators by providing:
- Multi-account management with login validation
- Draft creation and management
- AI-powered content polishing (text and image generation)
- Asset library for managing media files
- Trend monitoring and discovery
- Real-time mobile preview of posts
- MCP (Model Context Protocol) server integration for AI capabilities

## Target Users

Content creators, influencers, and marketers who regularly publish on Xiaohongshu and need efficient tools for managing multiple accounts and creating high-quality content.

## Key Features

- **Multi-Account Support**: Manage multiple Xiaohongshu accounts with secure login
- **Content Editor**: Create and edit posts with title, content, and images
- **AI Integration**: Text and image generation via configurable AI providers
- **Draft Management**: Save and organize drafts locally
- **Trend Discovery**: Monitor trending topics and content
- **Mobile Preview**: Real-time preview of how posts will appear on mobile devices
- **Asset Management**: Organize and select images for posts
- **MCP Server**: Built-in Model Context Protocol server for AI tool integration
